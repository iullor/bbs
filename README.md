## 项目题目：BBS学生论坛



### 项目介绍

​	校园的BBS论坛，方便大学校园学生的日常生活，快速了解学校的各种活动，信息

### 使用技术

- Java语言，基于BS架构的web应用
- SSM + MySQL
- 开发工具
